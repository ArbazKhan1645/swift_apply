# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-05

### Added
- Initial release of SwiftApply.
- Magic Fill (Clipboard intelligence) for automated data extraction.
- Dual Outreach Mode: SMTP Email & WhatsApp Business API.
- Integrated CV Library for managing multiple resumes.
- Local Job Lead tracking with SQLite.
- Premium Glassmorphism UI with Dark Mode support.
- Splash Screen and Onboarding flow.
- Deep linking support for LinkedIn and Indeed.
- Professional documentation (README, LICENSE, CONTRIBUTING).

### Fixed
- Fixed critical issues with modal sheet controller disposal.
- Optimized route pre-filling logic to prevent build-phase state updates.
- Resolved layout overflows in modal sheets when the keyboard is visible.
